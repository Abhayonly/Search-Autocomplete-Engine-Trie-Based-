# Search Autocomplete Engine (Trie-Based)

##  Overview
A C++ console-based autocomplete system built using the Trie data structure.

## Features
- Insert words into Trie
- Search words by prefix
- Generate dynamic suggestions
- Fast lookup with O(L) time complexity

## Tech Stack
- C++
- Object-Oriented Programming
- STL (vector, unordered_map)
- Recursion

## Concepts Used
- Trie Data Structure
- Prefix Searching
- DFS Traversal
